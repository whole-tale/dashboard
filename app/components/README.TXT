WholeTale Components
====================

Components have been rewritten to sort through the useful ones and ones that should be expired:

dashboard - contains the new UI for the dashboard implemented entirely using components and sub components.
ui - ui components that form active part of the UI, UI contains sub-directories for grouping of components e.g. files contains all things to do with managing files and so forth.
old - components that are no longer used and should be deleted at some point
